# Support

OpenLedger is local-first finance records software. Support is best-effort and focused on reproducible bugs, documentation, local development, import/export behavior, and privacy concerns.

## Before Opening An Issue

- Check the [README](README.md), [SECURITY.md](SECURITY.md), and [CONTRIBUTING.md](CONTRIBUTING.md).
- Search existing issues and pull requests.
- Do not share bank records, account numbers, API keys, private exports, or sensitive financial screenshots.

## Good Issue Reports

- Describe whether the issue affects import, export, budgeting, local storage, or setup.
- Include anonymized sample data only if it is safe to make public.
- Provide steps to reproduce and expected behavior.

For security concerns, follow [SECURITY.md](SECURITY.md).
